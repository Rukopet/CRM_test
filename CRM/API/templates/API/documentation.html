--------------------------------------------------------------------------------CLIENTS--------------------------------------------------------------------------------
localhost/clients/    --> вывод всех склиентов
localhost/clients/{int:id}/    --> вывод информации клиента с конкретным id
localhost/clients/{int:id}/delete/    --> удаление записи клиента с конкретным id
localhost/clients/{int:id}/change/{str:field}={str:value}/    --> изменение конретного поля
localhost/clients/create/{str:name}&{str:telegram_id}/    --> создание клиента связка имя & девятизначный telegram id или 0 или None
localhost/clients/create/    --> создание клиента через json форму (POST)

----------------------------------------------------------------------------------BIDS----------------------------------------------------------------------------------
localhost/bids/    --> вывод всех заявок
localhost/bids/{int:id}/    --> вывод информации по заявке с конкретным id
localhost/bids/{int:id}/delete/    --> удаление записи заявки с конкретным id
localhost/bids/{int:id}/change/{str:field}={str:value}/    --> изменение конретного поля заявки
localhost/bids/{int:type_bid}&{int:client_id}&{str:body}&{slug:title_bid}&{int:notifications}/    --> создание заявки через get запрос
localhost/bids/create/    --> создание заявки через json форму (POST)
localhost/bids/category/  --> сортировка по категориям, категории разделяются на заявки:
 по типам: "fix", "consultation", "service"
 по статусу:"open", "in work", "finished"
localhost/bids/{yyyy-mm-dd}/    --> вывод заявок за конкретную дату
localhost/bids/{yyyy-mm-dd}/{yyyy-mm-dd}    --> вывод заявок за пероид между датами(включительно)
